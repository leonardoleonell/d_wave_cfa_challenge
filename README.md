# D-Wave CFA Challenge

Financial analysis project for the D-Wave CFA Challenge.

## Project Structure

- `data/`: datasets used in the analysis
- `notebooks/`: exploratory analysis and modeling
- `report_sections/`: written report sections
- `slides/`: presentation materials
- `requirements.txt`: Python dependencies

## Objective

Analyze D-Wave's financial performance, revenue growth, profitability, and valuation assumptions.

## Tools

- Python
- Pandas
- Matplotlib
- Excel
- VS Code
