# Perplexity interim files

This folder is for **working files after standardization but before final validation**.

Typical contents:

- standardized annual income statement
- standardized annual balance sheet
- standardized annual cash flow statement
- standardized key stats / ratios / KPI files
- line-item mapping tables
- reconciliation working papers

Examples:

```text
qbts_income_statement_annual_standardized.csv
qbts_balance_sheet_annual_standardized.csv
qbts_cash_flow_annual_standardized.csv
qbts_line_item_mapping.csv
```

Files in this folder may:

- normalize labels
- convert units
- align fiscal years
- separate GAAP from adjusted measures
- prepare data for reconciliation

They are **not** yet the canonical source for modeling.
