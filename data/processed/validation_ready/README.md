# Validation-ready datasets

Use this folder for consolidated files that are ready for formal checks but have not yet been promoted to the final canonical cleaned dataset.

Example:

```text
qbts_financials_perplexity_validation_ready.csv
```

Only after reconciliation and validation should the final file be written to:

```text
data/processed/qbts_financials_perplexity_clean.csv
```
